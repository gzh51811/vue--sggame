# vue项目


### 杉果

> 介绍：在三阶段vue和nodejs的基础上实现一个手机端万展

### 项目情况-运用技术

*后端语言：使用的是Nodejs+Express

*数据库：使用的是MongoDB

*UI框架：使用element-ui框架 Minit-UI

*登陆使用token进行加密、解密、验证

*其他页面使用nodejs+Express和其他第三方插件
